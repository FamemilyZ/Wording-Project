## [2.0.0] - 2568-12-04

### เพิ่ม (Added)
- เพิ่มฟีเจอร์ใหม่ "Wheel Mode" สำหรับการสุ่มคำศัพท์:
  - เพิ่มปุ่ม "Wheel Mode" ที่มุมขวาบนของหน้าเว็บเพื่อเข้าสู่โหมดใหม่
  - สร้างหน้าจอ UI สำหรับ Wheel Mode ประกอบด้วยวงล้อ, ปุ่ม Spin, และส่วนจัดการคำศัพท์
  - ผู้ใช้สามารถนำเข้าคำศัพท์ได้ 2 วิธี:
    1. พิมพ์ข้อความโดยตรงในช่อง Text Area
    2. อัปโหลดไฟล์ `.txt` (รองรับการคลิกและ Drag & Drop)
  - ระบบจะแสดงรายการคำศัพท์ทั้งหมดที่นำเข้ามา
  - เมื่อกดปุ่ม "Spin" วงล้อจะหมุนและสุ่มเลือกคำศัพท์ 1 คำจากรายการ



## [1.65] - 2568-11-25

### เพิ่ม (Added)
- เพิ่มปุ่ม 'Back to My Levels' ในหน้าสร้างด่านเพื่อกลับไปยังหน้ารายการด่านที่สร้างเอง
- เพิ่มหน้าจอตัวเลือกก่อนเริ่มควิซเพื่อให้ผู้ใช้สามารถกำหนดจำนวนคำถามได้
- ผู้ใช้สามารถเลือกทำควิซด้วยคำศัพท์ทั้งหมดในด่านได้

### ปรับปรุง (Changed)
- ปรับตำแหน่งปุ่ม 'Back to Menu' ในหน้าทำควิซไปไว้มุมขวาล่าง
- ปรับแต่ง ui และจัดวางตำแหน่งใหม่เล็กน้อย
- ปรับปรุงเอฟเฟกต์แสดงผลตอบรับในหน้าควิซให้เป็นการเปลี่ยนสีทั้งหน้าจอเพื่อความชัดเจน
- ปรับปรุงขั้นตอนการเริ่มควิซใหม่ทั้งหมด โดยจะแสดงหน้าจอตัวเลือกก่อนเสมอ

## [1.6] - 2568-11-25

### เพิ่ม (Added)
- เพิ่มระบบยืนยันตัวตนผู้ใช้ (Authentication) โดยใช้ Firebase Authentication
- ผู้ใช้สามารถล็อกอินด้วยบัญชี Google หรือ GitHub ได้
- เพิ่มหน้าจอสำหรับล็อกอินโดยเฉพาะ
- เพิ่มปุ่ม Logout ในส่วนหัวของเว็บสำหรับผู้ใช้ที่ล็อกอินแล้ว
- จำกัดการเข้าถึงฟีเจอร์ของแอป (เช่น การทำควิซ) เฉพาะผู้ใช้ที่ล็อกอินเข้าระบบแล้วเท่านั้น
- เพิ่มระบบจัดการด่านที่สร้างเอง (Custom Level Management) ที่บันทึกข้อมูลแยกตามบัญชีผู้ใช้ผ่าน Cloud Firestore
- เพิ่มหน้า "ด่านของฉัน" (My Levels) สำหรับการดู, เล่น, แก้ไข, และลบด่านที่ผู้ใช้สร้างเอง
- เพิ่มความสามารถในการแก้ไข (เพิ่ม/ลบคำศัพท์) และลบด่านที่สร้างเองได้

### ปรับปรุง (Changed)
- ปรับปรุงดีไซน์ของหน้าจอ Login ทั้งหมดให้ดูทันสมัยและสะอาดตา (Minimalist) เพื่อประสบการณ์ใช้งานที่ดีขึ้น
- ปรับปรุงระบบด่านที่สร้างเอง (Custom Mode) ใหม่ทั้งหมด:
    - แยก "ด่านของฉัน" ออกจาก "โหมดเลเวล" (Level Mode) อย่างชัดเจน
    - เปลี่ยนขั้นตอนการเข้าถึง ให้ "Custom Mode" นำไปที่หน้า "ด่านของฉัน" ก่อน
    - ย้ายข้อมูลด่านที่สร้างเองจาก `localStorage` ไปยัง Cloud Firestore เพื่อให้สามารถใช้งานข้ามอุปกรณ์ได้
- ปรับปรุงข้อความแสดงข้อผิดพลาด (Error Messages) ในหน้าล็อกอินให้มีความเฉพาะเจาะจงและเป็นภาษาไทยที่เข้าใจง่ายมากขึ้น (เช่น กรณีไม่พบผู้ใช้, ลืมกรอกรหัสผ่าน)
- เพิ่มแอนิเมชัน (UI Animation) เมื่อกดปุ่มในหน้าเมนูหลัก (Mode Selection) เพื่อประสบการณ์ใช้งานที่ดีขึ้น
- ปรับปรุงแอนิเมชันการเปลี่ยนหน้าจอหลักให้เป็นแบบสไลด์ (Slide Animation) เพื่อเพิ่มความน่าสนใจในการใช้งาน

### แก้ไข (Fixed)
- แก้ไขข้อผิดพลาด `auth/unauthorized-domain` โดยการตั้งค่าโดเมนที่ได้รับอนุญาตใน Firebase console
- แก้ไขปุ่ม "Sign in with Google" ที่ไม่แสดงผลตรงกลางหน้าจอ
- แก้ไขการตัดคำที่ไม่สวยงามของข้อความแสดงข้อผิดพลาด
