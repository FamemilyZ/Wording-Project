# บันทึกการเปลี่ยนแปลง (Changelog)

การเปลี่ยนแปลงที่สำคัญทั้งหมดของโปรเจกต์นี้จะถูกบันทึกไว้ในไฟล์นี้

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5] - 2025-11-25

### ถูกนำออก (Removed)
- นำฟีเจอร์แสดงรูปภาพประกอบคำถามออกจากหน้าแบบทดสอบ เพื่อเพิ่มความท้าทายในการเล่น เนื่องจากทำให้ผู้ใช้คาดเดาคำตอบได้ง่ายเกินไป การเปลี่ยนแปลงนี้ครอบคลุมการแก้ไขไฟล์ HTML, CSS, และ JS เพื่อลบส่วนแสดงผลและ Logic ที่เกี่ยวข้องกับรูปภาพออกทั้งหมด
