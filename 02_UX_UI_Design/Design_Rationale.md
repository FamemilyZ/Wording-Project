# เอกสารอธิบายเหตุผลการออกแบบ (Design Rationale)

## 1. ปรัชญาการออกแบบ (UX Philosophy)
เรามุ่งเน้นการออกแบบที่ให้ประสบการณ์การใช้งานที่เรียบง่าย (Simplicity) และตรงไปตรงมา (Straightforward) เพื่อให้ผู้ใช้สามารถเข้าถึงฟังก์ชันหลัก (การเล่นเกมทายศัพท์) ได้อย่างรวดเร็วที่สุด โดยลดขั้นตอนที่ไม่จำเป็นและกำจัดสิ่งรบกวนต่างๆ ออกไป

## 2. การตัดสินใจด้านการออกแบบ (UI Choices)

### 2.1 โครงสร้างและ Layout
- **Mobile-First & Responsive Design:** ออกแบบโดยคำนึงถึงการใช้งานบนหน้าจอมือถือก่อน ทำให้โครงสร้างเป็นแบบคอลัมน์เดี่ยว (Single-Column) ที่ดูง่าย และสามารถปรับขยายไปแสดงผลบนหน้าจอเดสก์ท็อปได้อย่างลงตัว
- **Screen-Based Flow:** แบ่งหน้าการทำงานออกเป็นส่วนๆ ชัดเจน (เช่น หน้า Login, หน้าเมนู, หน้าเล่นเกม, หน้าผลลัพธ์) เพื่อให้ผู้ใช้เข้าใจลำดับการทำงานและไม่สับสน

### 2.2 ชุดสีและตัวอักษร (Color Palette & Typography)
- **ชุดสี:**
    - **สีหลัก (Primary):** สีน้ำเงิน (`#4285F4`) ใช้สำหรับปุ่มหลักที่ต้องการกระตุ้นให้ผู้ใช้กด เช่น "Sign In", "Start Quiz" เพื่อสร้างความรู้สึกน่าเชื่อถือและนำสายตา
    - **สีรอง (Secondary):** สีเทา (`#6c757d`) ใช้สำหรับปุ่มทางเลือกอื่นๆ เช่น "Sign Up" เพื่อลดความเด่นลงจากปุ่มหลัก
    - **สีพื้นหลัง:** สีขาวและเทาอ่อน (`#f8f9fa`) เพื่อให้ดูสะอาดตาและสบายตา ทำให้เนื้อหาและองค์ประกอบหลักดูโดดเด่น
    - **สีแจ้งเตือน:** สีเขียวสำหรับสถานะสำเร็จ (Success) และสีแดงสำหรับข้อผิดพลาด (Error) ซึ่งเป็นสีสากลที่ผู้ใช้เข้าใจได้ทันที
- **ตัวอักษร (Typography):**
    - เลือกใช้ฟอนต์ตระกูล Sans-serif (เช่น Roboto, Lato, หรือฟอนต์พื้นฐานของระบบ) เนื่องจากมีความ خوانง่ายบนหน้าจอดิจิทัลทุกขนาด

## 3. ขั้นตอนการทำงานของผู้ใช้ (User Flow)
ขั้นตอนการทำงานหลักถูกออกแบบมาให้สั้นและเข้าใจง่ายที่สุด:

1.  **เริ่มต้น:** ผู้ใช้เข้าสู่หน้า **Login Screen**
2.  **เข้าสู่ระบบ:** ผู้ใช้ทำการล็อกอินหรือลงทะเบียน
3.  **หน้าเมนูหลัก:** เมื่อล็อกอินสำเร็จ จะเข้าสู่หน้า **Mode Screen** เพื่อเลือกโหมดการเล่น (Level Mode / Custom Mode)
4.  **เลือกด่าน:** ผู้ใช้เลือกด่านที่ต้องการเล่น
5.  **เล่นเกม:** เข้าสู่หน้า **Quiz Screen** เพื่อทำแบบทดสอบ
6.  **ดูผลลัพธ์:** เมื่อจบเกม จะเข้าสู่หน้า **Result Screen** เพื่อดูคะแนน
7.  **กลับไปเริ่มต้น:** ผู้ใช้สามารถเลือกกลับไปที่เมนูหลักเพื่อเล่นต่อได้
