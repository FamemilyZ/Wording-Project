# คู่มือการใช้งานโปรแกรม Wording

---

## 1. บทนำ

Wording เป็นเว็บแอปพลิเคชันที่ออกแบบมาเพื่อช่วยให้คุณฝึกฝนและเรียนรู้คำศัพท์ภาษาอังกฤษผ่านเกมทายศัพท์ที่แบ่งตามระดับความยากง่าย สามารถเข้าใช้งานได้ที่: **[https://wording-f2155.web.app](https://wording-f2155.web.app)**

ผู้ใช้สามารถลงทะเบียนเพื่อบันทึกความคืบหน้าและแข่งขันกับตัวเองได้

## 2. การเริ่มต้นใช้งาน

### 2.1 การลงทะเบียน
1.  ไปที่หน้าแรกของแอปพลิเคชัน
2.  คลิกที่ปุ่ม "ลงทะเบียน" หรือ "Sign Up"
3.  กรอกอีเมลและตั้งรหัสผ่านของคุณ จากนั้นกดยืนยัน
4.  เมื่อลงทะเบียนสำเร็จ ระบบจะนำคุณไปยังหน้าเข้าสู่ระบบ

### 2.2 การเข้าสู่ระบบ
1.  ไปที่หน้าแรกของแอปพลิเคชัน
2.  กรอกอีเมลและรหัสผ่านที่คุณได้ลงทะเบียนไว้
3.  คลิกที่ปุ่ม "เข้าสู่ระบบ" หรือ "Login"

## 3. ฟังก์ชันหลัก

### 3.1 หน้าหลัก
หลังจากเข้าสู่ระบบ คุณจะพบกับหน้าหลักซึ่งประกอบด้วย:
*   **เมนูเลือกระดับ:** สำหรับเลือกด่านความยากของคำศัพท์
*   **ข้อมูลผู้ใช้:** แสดงชื่อและคะแนนสะสมของคุณ

### 3.2 วิธีการเล่น
1.  จากหน้าหลัก ให้เลือกระดับ (Level) ที่คุณต้องการเล่น
2.  เมื่อเกมเริ่ม ระบบจะแสดงคำศัพท์ขึ้นมา
3.  ให้คุณพิมพ์คำตอบที่ถูกต้องลงในช่องว่างและกด "Enter" หรือปุ่ม "ส่งคำตอบ"
4.  ระบบจะตรวจคำตอบและแสดงผลว่าถูกหรือผิด พร้อมทั้งอัปเดตคะแนนของคุณ
5.  เล่นต่อไปจนกว่าจะจบด่าน

## 4. คำถามที่พบบ่อย

**ถาม:** ลืมรหัสผ่านต้องทำอย่างไร?
**ตอบ:** ขณะนี้ฟังก์ชันกู้คืนรหัสผ่านยังอยู่ในระหว่างการพัฒนา กรุณาติดต่อผู้ดูแลระบบ

**ถาม:** หน้าเว็บไม่โหลดหรือค้าง ควรทำอย่างไร?
**ตอบ:** ลองทำการรีเฟรชหน้าเว็บ (กด F5 หรือปุ่ม Refresh) หรือตรวจสอบการเชื่อมต่ออินเทอร์เน็ตของคุณ
