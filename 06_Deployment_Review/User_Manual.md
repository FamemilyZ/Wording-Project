# คู่มือการใช้งานโปรแกรม Wording

## 1. บทนำ
Wording เป็นเว็บแอปพลิเคชันที่ออกแบบมาเพื่อช่วยให้คุณฝึกฝนและเรียนรู้คำศัพท์ภาษาอังกฤษผ่านเกมทายศัพท์ที่แบ่งตามระดับความยากง่าย สามารถเข้าใช้งานได้ที่: **[https://wording-f2155.web.app](https://wording-f2155.web.app)**

ผู้ใช้สามารถลงทะเบียนเพื่อบันทึกความคืบหน้าและแข่งขันกับตัวเองได้

## 2. การเริ่มต้นใช้งาน

### 2.1 การลงทะเบียน
1.  ไปที่หน้าแรกของแอปพลิเคชัน
2.  คลิกที่ปุ่ม "Sign Up"
3.  กรอกอีเมลและตั้งรหัสผ่านของคุณ จากนั้นกดยืนยัน
4.  เมื่อลงทะเบียนสำเร็จ ระบบจะนำคุณไปยังหน้าเข้าสู่ระบบ

### 2.2 การเข้าสู่ระบบ
1.  ไปที่หน้าแรกของแอปพลิเคชัน
2.  กรอกอีเมลและรหัสผ่านที่คุณได้ลงทะเบียนไว้
3.  คลิกที่ปุ่ม "Sign In" หรือเข้าสู่ระบบผ่าน Google

## 3. ฟังก์ชันหลัก

### 3.1 การเลือกโหมด
หลังจากเข้าสู่ระบบ คุณจะพบกับหน้าจอให้เลือกโหมด 2 รูปแบบ:
- **Level Mode:** เล่นเกมจากด่านมาตรฐานที่มีคำศัพท์มาให้
- **Custom Mode:** สร้างและเล่นด่านคำศัพท์ของคุณเอง

### 3.2 วิธีการเล่น (Level Mode)
1.  จากหน้าเลือกโหมด คลิก "Level Mode"
2.  เลือกระดับ (Level) ที่คุณต้องการเล่น
3.  ระบุจำนวนคำถามที่ต้องการ หรือกด "All Words" เพื่อเล่นทั้งหมด
4.  เมื่อเกมเริ่ม ระบบจะแสดงคำศัพท์และตัวเลือก
5.  คลิกเลือกคำตอบที่ถูกต้อง
6.  เมื่อจบเกม ระบบจะแสดงหน้าจอผลลัพธ์พร้อมคะแนนของคุณ

### 3.3 วิธีการสร้างและเล่นด่านของตัวเอง (Custom Mode)
1.  จากหน้าเลือกโหมด คลิก "Custom Mode"
2.  คุณจะเห็นรายการด่านที่เคยสร้างไว้ (ถ้ามี) คลิก "Create Custom Level" เพื่อสร้างด่านใหม่
3.  ตั้งชื่อด่าน และเริ่มพิมพ์คำศัพท์ที่ต้องการ (อังกฤษหรือไทย) แล้วกด "Translate and Add"
4.  เมื่อเพิ่มคำศัพท์อย่างน้อย 4 คำแล้ว กด "บันทึกด่าน"
5.  คุณสามารถเลือกเล่นด่านที่สร้างไว้ได้จากหน้ารายการด่านของฉัน

### 3.4 ประวัติการเล่น
- จากหน้าเลือกโหมด สามารถกดปุ่ม "View History" เพื่อดูประวัติและคะแนนการเล่นย้อนหลังได้

## 4. คำถามที่พบบ่อย
**ถาม:** ลืมรหัสผ่านต้องทำอย่างไร?
**ตอบ:** ระบบของ Firebase มีมาตรฐานความปลอดภัยสูง หากลืมรหัสผ่าน กรุณาใช้ฟังก์ชัน "ลืมรหัสผ่าน" (ถ้ามี) หรือติดต่อผู้พัฒนา

**ถาม:** หน้าเว็บไม่โหลดหรือค้าง ควรทำอย่างไร?
**ตอบ:** ลองทำการรีเฟรชหน้าเว็บ (กด Ctrl+R) หรือตรวจสอบการเชื่อมต่ออินเทอร์เน็ตของคุณ
